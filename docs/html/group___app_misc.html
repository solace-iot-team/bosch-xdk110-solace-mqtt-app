Create a topic string according to the template.

The template specifies the format and the parameters the substitutions in the template.

Parameters
[in]templatethe snprintf template, e.g. "%s/iot-control/%s/device/%s/command"
[in]methodthe first substitution
[in]baseTopicthe second substitution
[in]deviceIdthe third substitution
Returns
char *: the topic created with malloc()

Example Usage:

char * topic = AppMisc_FormatTopic("%s/iot-control/%s/device/%s/command", methodCreate, baseTopic, appCmdCtrl_DeviceId);

void AppMisc_UserFeedback_InSetup ( void  )

Sets the LEDs into rolling mode to indicate to user that device is in 'set-up'/'connecting' mode.

Note
Setup and enable LEDs before calling.

Example Usage:

if (RETCODE_OK == retcode) retcode = LED_Setup();
if (RETCODE_OK == retcode) retcode = LED_Enable();
if (RETCODE_OK == retcode) AppMisc_UserFeedback_InSetup();
