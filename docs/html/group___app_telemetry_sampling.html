◆ appTelemetrySampling_TelemetrySamplingTask()

static void appTelemetrySampling_TelemetrySamplingTask ( void *  pvParameters)
static

The sampling task. Runs a loop with a delay of the configured sampling interval. Reads the sensor data, creates a new AppTelemetryPayload and adds it to the AppTelemetryQueue. Updates the stats if sampling is slower than expected interval using AppStatus_Stats_IncrementTelemetrySamplingTooSlowCounter().

Parameters
[in]pvParametersunused.
