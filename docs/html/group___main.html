int main ( void  )

The main() function invoked at start-up.

Initializes the Retcode module with AppStatus_ErrorHandlingFunc()

Innitializes the command processor for the AppController module, enqueues the AppController_Init() function, and starts the scheduler.

Exceptions
mainRetcode_RaiseError() if AppController_Init() could not be enqueued.
mainassert(false) if scheduler could not be started.
