◆ AppTimestamp_CreateTimestampStr()

char* AppTimestamp_CreateTimestampStr ( AppTimestamp_T  timestamp)

Returns the formatted timestamp string for the timestamp.

If timestamp.isTickCount==true, it calculates the past time.

See also
APP_TIMESTAMP_STRING_FORMAT
Parameters
[in]timestampthe timestamp generated with AppTimestamp_GetTimestamp()
Returns
char *: the timestamp converted to string or NULL if module is not enabled yet.
Note
Returned char * must be deleted after use.

Example Usage:

// lock in the timestamp
AppTimestamp_T savedTimestamp = AppTimestamp_GetTimestamp(xTaskGetTickCount());
// do something in between
char * timestampStr = AppTimestamp_CreateTimestampStr(savedTimestamp);
// do something
free(timestampStr);

◆ AppTimestamp_Enable()

Retcode_T AppTimestamp_Enable ( void  )

Enable the module.

Enables the SNTP module, contacts the SNTP server, retrieves the current time and uses current tick counts as the baseline. Tries APP_TIMESTAMP_NUM_SNTP_TRIES times before it gives up. Call after AppTimestamp_Setup().

Returns
Retcode_T: RETCODE_OK
Retcode_T: RETCODE(RETCODE_SEVERITY_FATAL, RETCODE_SOLAPP_FAILED_TO_SYNC_SNTP_TIME_FROM_SERVER)

◆ AppTimestamp_GetTimestamp()

AppTimestamp_T AppTimestamp_GetTimestamp ( const TickType_t  tickCount)

Returns the timestamp of tickCount.

If module is enabled:

timestamp.secondsSinceEpoch & timestamp.millis are set and timestamp.isTickCount = false

If module is not enabled:

timestamp.isTickCount = true and timestamp.tickCount = tickCount

Parameters
[in]tickCountthe tick count for which to calculate the timestamp from.
Returns
AppTimestamp_T: the timestamp

Example Usage:

AppTimestamp_T currentTimestamp = AppTimestamp_GetTimestamp(xTaskGetTickCount());
if(currentTimestamp.isTickCount) {
// not enabled yet
} else {
// enabled
}

wrong use, to capture past time use AppTimestamp_T, not tickCount
