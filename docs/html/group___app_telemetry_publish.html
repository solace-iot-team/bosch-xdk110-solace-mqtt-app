◆ AppTelemetryPublish_ApplyNewRuntimeConfig()

Retcode_T AppTelemetryPublish_ApplyNewRuntimeConfig ( AppRuntimeConfig_ConfigElement_T  configElement,
const void *  newConfigPtr 
)

Applies a new runtime config to the module.

Extracts the following, depending on configElement

AppRuntimeConfig_Element_targetTelemetryConfig: qos

AppRuntimeConfig_Element_topicConfig: creates the new topic

AppRuntimeConfig_Element_activeTelemetryRTParams: the publishing frequency

Note
Call only if publishing task is not running.
Parameters
[in]configElement: the type of config to apply.
[in]newConfigPtr: the pointer, must match the type. Note: no type checking performed.
Returns
Retcode_T: RETCODE_OK
Retcode_T: RETCODE(RETCODE_SEVERITY_FATAL, RETCODE_SOLAPP_UNSUPPORTED_RUNTIME_CONFIG_ELEMENT)
Retcode_T: RETCODE(RETCODE_SEVERITY_FATAL, RETCODE_SOLAPP_FAILED_TO_TAKE_SEMAPHORE_IN_TIME)
